# 📊 Retail Sales Analytics using Python

## 📌 Project Overview

This project analyzes retail sales data using Python to uncover valuable business insights related to sales performance, profitability, customer behavior, product performance, regional trends, and shipping analysis.

The objective is to demonstrate how Python can be used for data cleaning, exploratory data analysis (EDA), business reporting, and data visualization.

---

## 🎯 Business Objectives

- Analyze overall sales and profit performance
- Identify top-performing categories and products
- Discover loss-making products
- Analyze customer purchasing behavior
- Evaluate regional sales performance
- Understand the relationship between discounts and profit
- Generate actionable business insights

## 💡 Key Business Insights

- Technology generated the highest sales and profit.
- Furniture showed high sales but comparatively lower profitability.
- Several products consistently generated losses.
- Higher discounts generally reduced profitability.
- The West region recorded the highest overall profit.
- Standard Class was the most frequently used shipping mode.
- Sales showed noticeable monthly fluctuations indicating seasonal demand.
